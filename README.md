# Only Human - Holding Page

A simple coming soon page for Only Human.

## Hosting

This is a static HTML page that can be hosted on GitHub Pages or any static hosting service.

To deploy on GitHub Pages:
1. Push this repository to GitHub.
2. Go to repository settings.
3. Enable GitHub Pages and select the main branch.
